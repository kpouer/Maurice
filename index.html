Maurice
GitHub
Maurice is a Thomson MO5 emulator written in Rust.
It is a port of Marcel o Cinq MO5 Emulator (Java version). Of course, it is opensource, and you can browse them at https://github.com/kpouer/Maurice
You can drop a tape file (MO5 was mostly using tapes, usually named .k7) then type
  • LOAD
  • RUN
